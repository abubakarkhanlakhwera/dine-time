# Dine Time Mobile App

An Expo mobile app experiment for dining-time or scheduling workflows.

## Purpose

Dining and reservation-style experiences need quick mobile flows for browsing, timing, and user decisions. This repo explores that kind of mobile app structure with Expo.

## What This Shows

- Expo app structure
- Mobile UI workflow practice
- JavaScript/React Native learning

## Stack

- Expo
- React Native
- JavaScript
